# Internship_project
# 📄 Resume Classification System

An end-to-end Machine Learning project that classifies resumes into different job categories using Natural Language Processing (NLP) techniques. The system transforms unstructured resume text into meaningful structured features and predicts the appropriate category.

---

## 📌 Project Overview

This project focuses on automating resume screening by classifying resumes into predefined categories. It uses NLP preprocessing, feature engineering, and multiple machine learning models to achieve high accuracy.

## 🧹 NLP Preprocessing

- Converted unstructured resume text into structured format  
- Applied text cleaning and normalization  
- Performed tokenization  
- Implemented word-count–based feature engineering  

---

## 📊 Exploratory Data Analysis (EDA)

- Analyzed category distribution  
- Studied keyword frequency  
- Examined resume length statistics  
- Created text visualizations  

---

## 🔢 Feature Engineering

- Converted text into numerical features using **TF-IDF Vectorization**  
- Extracted meaningful patterns from textual data  

---

## 🤖 Model Training

Trained and evaluated multiple machine learning models:
- Decision Tree  
- Random Forest  
- Support Vector Machine (SVM)  
- K-Nearest Neighbors (KNN)  
- Gradient Boosting  
- XGBoost  

✅ Achieved **~90% accuracy with Random Forest**

---

## 📈 Model Evaluation

- Compared performance across multiple models  
- Selected best model based on accuracy and performance metrics  

---

## 🌐 Deployment

- Built an interactive web application using **Streamlit**  
- Enables real-time resume classification
