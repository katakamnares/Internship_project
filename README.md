# Internship_project
Developed an end-to-end resume classification system using NLP. Preprocessed resume text (cleaning, tokenization) and performed EDA on categories, keywords, and resume length. Converted text to features using TF-IDF and trained ML models; Random Forest achieved ~90% accuracy. Deployed a real-time Streamlit app for classification.
